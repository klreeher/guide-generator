Guides · Guidelines

Requirements

In order for your Google Docs guide to be correctly parsed by this generator, you will need to ensure it is correctly formatted. You guide must:

  • Have at least one title (with h1, h2, h3, h4 or h5 heading) at the very top before any paragraphs
  • Any text you want interpreted as code will need to be colored red. Hover over the colors in the color palette and makes sure its plain red and not some other shade
  • No google docs comments can be visible, they should be resolved or closed

Known Issues

  • There can only be one code block per section - working to fix this
