Introduction

This guide will teach you how to set different prices for the same products based on different users.

In B2B commerce, it is common that the same product is sold to different customers at different price points. This is often referred to as multi-channel commerce. There are many different reasons a seller may want to vary the price of a product (location, volume, relationship, contractual agreement, etc.).

As an example, you may have two regions that buy the same product from you. The East region is a high volume purchaser and gets a lower price of $25/each for a product. The West region is a low volume purchaser and has an agreed price of $30/each for the same product.

1. Create a Product

Start by Creating one product that is sold in both the East and West region. To learn how to do this, reference this guide.

Create Two User Groups

Create one User Group called EastRegion, then create another User Group called WestRegion. To learn how to do this, reference this guide.

Create Two Price Schedules

Create one price schedule at a $25.00 price point with an ID of “PS-25”. Create a second price schedule at a $30.00 price point with an ID of “PS-30”. To learn how to do this, reference this guide.

Make the Assignments

Assign the product to the $25.00 price schedule and the EastRegion user group. Next, assign the same product to the $30.00 price schedule and the WestRegion user group. See the request body example below for how this will look. To learn how to make assignments, reference this guide.

What Ifs

  • What if I need different prices per buyer company? Prices schedules can be assigned at the Buyer, User Group, or User level, so you can have different prices for any of those parties. You will just need to create multiple price schedules and assign them to each desired party.
  • What if I need more than two prices? A product can have an unlimited number price schedules assigned to it.

    Conclusion

    At this point, you should know how to set-up personalized pricing by associating one product with different prices for different users.
