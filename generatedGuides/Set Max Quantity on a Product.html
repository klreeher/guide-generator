Introduction

This guide will teach you how to set a maximum line item quantity on a product. This can be used in any situation where you want to limit the maximum quantity a user can purchase for a single line item.

For example, in my store I want to enable users to select the quantity to be added to cart but restrict them from ordering more than 2,500 of a selected product.

Prerequisites

  • Create and Assign a Product
  • Create and Assign a Price Schedule
  • Set Minimum and Maximum Order Quantities

1. Create a Price Schedule

Assuming you have already created the product that you want to set a maximum quantity on, next you will create a corresponding Price Schedule.

When creating the Price Schedule, set the MaxQuantity value to the desired quantity limit. In this example, set it to 2500.

API Reference: Create a Price Schedule

POST https://api.ordercloud.io/v1/priceschedules/{priceScheduleID}s HTTP/1.1
Authentication: Bearer put_access_token_here
Content-Type: application/json; charset=UTF-8


{
 "ID": "ExamplePriceScheduleID",
 "Name": "Example Price Schedule Name",
 "ApplyTax": false,
 "ApplyShipping": false,
 "MinQuantity": null,
 "MaxQuantity": 2500,
 "UseCumulativeQuantity": false,
 "RestrictedQuantity": false,
 "OrderType": "Standard",
 "PriceBreaks": [
   {
     "Quantity": 1,
     "Price": 5.0
   }
 ],
 "xp": null
}

What Ifs

What if you need to assign this max quantity to all users?
You would need to assign the Price Schedule with the set MaxQuantity at the buyer level.

What if you need to assign different maximum quantities to different sets of users?
You would** have to create multiple Price Schedules with a unique MaxQuantity for each set of users needed. Then make sure the Price Schedule assignments correspond to the users or user groups you want.

What if you need to control the max quantity across the whole order for a specific product?
If you want to restrict the user from adding multiple line items of the same product in order to exceed the max quantity, set the UseCumulativeQuantity property to true on the Price Schedule. This will limit the user to specific quantities across their entire order.

Conclusion

This guide allows you to establish a maximum quantity on a product through a Price Schedule.
